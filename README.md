# Dentist app

## Quick start
### Backend
    
    cd backend
    npm i
    npm run build
    cp .env.example .env
    npm run start:docker
    
### Frontend

    cd frontend
    yarn install
    yarn start



## Open API documentation

Run the server inside docker for example and call the /doc route:

[http://localhost:4000/doc](http://localhost:4000/doc)
